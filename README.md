# Swift Gift

## Start

`yarn install`

`yarn start`

or

`npm i`

`npm start`

## Test

`yarn test`

or 

`npm run test`

## Used

- create-react-app (for fast start)
- axios (useful library for async requests)
- redux (for data layer)
- react-router (convenient router)
- styled-components (js css style)
